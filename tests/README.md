These are tests for j-pet-ortho-simulations repository.
They are based on a google test framework.

### Requirements:
+ Google test framework

### Installation under Ubuntu:
`sudo apt-get install libgtest-dev`
`cd /usr/src/gtest`
`cmake CMakeList.txt`
`make`
`cp *.a /usr/lib`
